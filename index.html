Laptop Showcase

Laptop Showcase is a simple, leightweigt, yet engaging way to show off your digital content such as websites, interface designs, and artwork in its intended context: a computer screen.

Features

  • Extremely Light (3kb)
  • Designed For Cross-Broser Usage (Compatibility Checks)
  • Responsive (works on desktops & mobile)
  • Flexible (showcase any content such as images, videos & iFrames)
  • Animated using CSS only (no JavaScript or jQuery)
    • Laptop Lid movement on Hover (mouse over)
    • Power Indicator pulses (inidcate standby) when not in Hover-State (mouse out). It then stops pulsing on Hover, indicating active use like on a real computer.
  • Responsive (Fluid & Bootstrap Breakpoints )

Usage

Requirements To Enable Latop Showcase

Laptop Showcase comes with 2 CSS files: one to provide base functionality such as the layout and computer design and the other to enable animations. The only necessary one is the laptopshowcase-base.css file. 

<!-- BASE: Necessary -->
<link rel="stylesheet" type="text/css" href="./laptop-showcase/laptopshowcase-base.css">
<!-- HOVER: Only necessary when using animations -->
<link rel="stylesheet" type="text/css" href="./laptop-showcase/laptopshowcase-hover.css">

HTML Code To Display A Laptop Showcase

<div class="laptop-showcase hover" id="preview">
	<div class="laptop" style="width:50%; [DEFINE YOUR SIZE HERE]">
				
		<div class="screen">
			<div class="screen-inner">
				[PLACE YOUR CONTENT TO SHOWCASE HERE]
			</div>
		</div>

		<div class="body">
			<div class="grip"></div>
			<div class="power"></div>
		</div>

	</div>
</div>
